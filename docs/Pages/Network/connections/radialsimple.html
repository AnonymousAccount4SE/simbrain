Radial (Simple) Connection

A radial (simple) connection will connect a given neuron to all neurons within a certain radius with a certain probability. The probability of forming inhibitory and excitatory connections are set separately as well as the synapse type associated with each type of connection. If a set of neurons are selected then connections will be made with the specified probabilities from each individual neuron, as when only one is selected.

The Simple Radial connection type connects neurons in succession from a starting point. This creates weights in a directed manner and activation radiates outward from the starting point or initial site of activation. This type of radial connection is "simple' as it is the most basic radial pattern of activation.

Connection Properties

Connection Methods

Deterministic: In a deterministic model, the cause of activity is predetermined and does not include elements of randomness.

Probabilistic: In a probabilistic model, the cause of activity is purely probabilistic and is based on elements of randomness.

Other Properties

Inward/Outward: Determines whether the path of activation radiates inwards or outwards. The outward property will connect each neuron to all the neurons directly surrounding it.

Exc. Radius: The distance to search for excitatory neurons to connect to. This has a minimum value of 0.

Inh. Radius: The distance to search for inhibitory neurons to connect to. This has a minimum value of 0.

Exc. Probability: The probabbility that connections will be made to neighbor excitatory (or non-polar) neurons. This has a minimum value of 0 and a maximum value of 1.

Inh. Probability: The probability that connections will be made to neighbor inhibitory neurons. This has a minimum value of 0 and a maximum value of 1.

Num. Exc. Connections: The maximum number of connections with excitatory neurons. This has a minimum value of 0.

Num. Inh. Connections: The maximum number of connections with inhibitory neurons. This has a minimum value of 0.
