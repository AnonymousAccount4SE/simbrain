Radial Connection


Radial connection will connect a given neuron to all neurons within a certain radius with a certain probability. The probability of forming inhibitory and excitatory connections are set separately as well as the synapse type associated with each type of connection. If a set of neurons are selected then connections will be made with the specified probabilities from each individual neuron, as when only one is selected.

There are two types of radial connections in Simbrain.

Radial (Simple)

This connection type connects neurons in succession from a starting point. This creates weights in a directed manner and activation radiates outward from the starting point or initial site of activation. This type of radial connection is termed “simple” as it is the most basic radial pattern of activation.

Radial

This connection type is like the Radial (Simple) type, but with more variations in how synapses radiate outward. This pattern is less predictable the the simple model.
